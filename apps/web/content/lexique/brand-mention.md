## Qu'est-ce qu'une brand mention ?

Une brand mention est une citation de votre marque dans une réponse IA. Quand ChatGPT répond "Pour la gestion de projet, Notion et Asana sont deux options populaires", Notion et Asana reçoivent chacune une brand mention.

Ces mentions sont l'unité de base du GEO. Avant de mesurer le Share of Voice ou la position, il faut déjà être mentionné.

## Types de brand mentions

### Mention explicite

Votre nom de marque est clairement cité : "Slack est une solution de communication d'équipe populaire."

### Mention contextuelle

Vous êtes cité dans un contexte spécifique : "Pour les startups tech, Slack est souvent recommandé."

### Mention comparative

Vous apparaissez dans une comparaison : "Slack vs Microsoft Teams : les deux ont leurs avantages."

### Mention négative

Vous êtes cité mais avec des réserves : "Slack peut être coûteux pour les grandes équipes."

## Pourquoi les brand mentions comptent

### Condition préalable à la visibilité

Sans mention, pas de visibilité. Les brand mentions sont le minimum requis pour exister dans l'écosystème IA.

### Signal de notoriété

Les LLM mentionnent les marques qu'ils "connaissent" — celles présentes dans leurs données d'entraînement avec une fréquence et un contexte suffisants.

### Base du calcul des métriques

Le Share of Voice, le taux de citation, et la position moyenne sont tous calculés à partir des brand mentions. Plus de mentions = plus de données = métriques plus fiables.

## Facteurs qui influencent les brand mentions

### Volume de contenu

Une marque avec beaucoup de contenu en ligne (site, blog, PR, avis) a plus de chances d'être présente dans les données d'entraînement.

### Qualité des sources

Les mentions dans des sources de qualité (presse reconnue, sites d'autorité) comptent plus que les mentions dans des sources obscures.

### Récurrence des associations

Une marque systématiquement associée à une catégorie sera plus facilement mentionnée pour cette catégorie.

### Récence

Pour les modèles avec accès web, le contenu récent peut influencer les mentions. Pour les modèles sans accès web, les données d'entraînement ont une date limite.
