## Qu'est-ce que le knowledge cutoff ?

Le knowledge cutoff est la date à laquelle les données d'entraînement d'un LLM s'arrêtent. Si un modèle a un cutoff au 1er janvier 2025, il ne "sait" rien de ce qui s'est passé après cette date.

Pour le GEO, cela signifie que les événements, lancements de produits, ou mentions récentes ne sont pas pris en compte par les modèles sans accès web.

## Impact du knowledge cutoff sur la visibilité

### Marques récentes

Une startup lancée après le cutoff d'un modèle est potentiellement inconnue de ce modèle. Elle sera moins recommandée car absente des données d'entraînement.

### Évolutions de marque

Un repositionnement, un nouveau produit, ou une amélioration significative après le cutoff ne sera pas reflété dans les réponses.

### Actualité du secteur

Les changements de dynamique concurrentielle post-cutoff ne sont pas pris en compte. Un concurrent en déclin peut toujours être recommandé, un nouvel entrant peut être ignoré.

## Stratégies face au knowledge cutoff

### Privilégier les modèles avec accès web

ChatGPT avec browsing, Perplexity, et d'autres peuvent accéder au web en temps réel. Votre contenu récent y est visible.

### Anticiper les mises à jour

Les modèles sont régulièrement mis à jour avec de nouvelles données. Publier du contenu maintenant prépare les futures versions.

### Vérifier les informations

Si un LLM donne des informations obsolètes sur votre marque, c'est probablement dû au cutoff. Les modèles avec accès web corrigeront cela.

## Knowledge cutoff des principaux LLM

| Modèle     | Cutoff approximatif | Accès web           |
| ---------- | ------------------- | ------------------- |
| GPT-4o     | Récent              | Oui (avec browsing) |
| Claude 3.5 | Variable            | Via intégrations    |
| Gemini     | Récent              | Oui                 |

Les cutoffs évoluent avec les mises à jour. Consultez la documentation officielle pour les dates exactes.

## Implications pour le GEO

Le knowledge cutoff est une raison supplémentaire de maintenir une stratégie GEO continue. Votre visibilité dans les modèles actuels influence les futures versions, et les modèles avec accès web reflètent votre présence actuelle.
