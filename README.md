# Coucou IA

Marketing site for Coucou IA, an AI consulting company for the French market. See `CLAUDE.md` for the working contract and `docs/design-system.md` for the design spec.

## Commands

```bash
pnpm dev          # local dev
pnpm build        # production build
pnpm lint         # eslint
pnpm typecheck    # tsc --noEmit
```
